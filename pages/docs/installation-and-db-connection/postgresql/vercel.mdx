# Vercel Postgres

According to their [official website](https://vercel.com/docs/storage/vercel-postgres),
Vercel Postgres is a is a serverless SQL database designed to integrate with Vercel Functions.

Drizzle ORM natively supports both [@vercel/postgres](https://github.com/neondatabase/serverless) serverless
driver with `drizzle-orm/vercer-postgres` package and [`postgres`](./postgresjs) or [`pg`](./node-postgres)
drivers to access Vercel Postgres through `postgesql://`

Checkout official [Vercel Postgres + Drizzle](https://vercel.com/docs/storage/vercel-postgres/using-an-orm#drizzle) docs

### Prepare Vercel Postgres

Setup project according to their [official docs](https://vercel.com/docs/storage/vercel-postgres/quickstart)  

### Make your first query

```typescript copy filename="index.ts"
import { sql } from '@vercel/postgres';
import { drizzle } from 'drizzle-orm/vercel-postgres';

const db = drizzle(sql)
const result = await db.select().from(...);
```
</Steps>

With [@vercel/postgres](https://github.com/neondatabase/serverless) severless package
you can access Vercel Postgres from either serverful or serverless environments with no TCP available,
like Cloudflare Workers, through websockets.  
  
If you're about to use Vercel Postgres from a serverfull environments - you can do it
either with `@vercel/postgres` package or access database through `postgesql://` protocol
with either [`postgres`](./postgresjs) or [`pg`](./node-postgres)
