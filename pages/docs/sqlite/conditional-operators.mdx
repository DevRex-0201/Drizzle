```typescript
eq(column, value)
eq(column1, column2)

notEq(column, value)

gt(column, value)

gte(column, value)

isNull(column)

```

### eq

Value equals `n`

#### Column to value

Column equals value

```typescript
db.select()
  .from(table)
  .where(eq(table.integerColumnName, 5))
  .all();
```

```sql
SELECT * 
  FROM table_name 
  WHERE table_name.integer_column_name = 5
```

#### Column to column

Column equals column

```typescript
db.select()
 .from(table)
 .leftJoin(table2, eq(table.integerColumnName, table2.foreign_integer))
 .all();
```

```sql
SELECT * 
  FROM table_name 
  LEFT JOIN table_name2 
  ON table_name.integer_column_name = table_name2.foreign_integer
```

### inArray

#### Column to values

```typescript

  .from(table)
  .where(inArray(table.integerColumnName, [1, 2, 3, 4]))
  .all();
```

```sql
SELECT *
  FROM table_name 
  WHERE table_name.integer_column_name in (1, 2, 3, 4)
```

#### Column to subquery

```typescript
db.select()
  .from(table)
  .where(
    inArray(
    table.string_column,
    db.select({ data: table2.string_column }).from(table2)
    )
  ).all();
```

```sql
SELECT * 
  FROM table_name 
  WHERE table_name.text_column_name 
  IN (SELECT table_name2.text_column_name FROM table_name2)
```

### not

All conditions must return `false`.

```typescript
db.select()
  .from(table)
  .where(not(eq(table.integerColumnName, 5)))
  .all();
```

```sql
SELECT * 
  FROM table_name 
  WHERE NOT (table_name.integer_column_name = 5)
```

### and

All conditions must return `true`.

```typescript
db.select()
 .from(table)
 .where(and(gt(table.integerColumnName, 5), lt(table.integerColumnName, 7)))
 .all();
```

```sql
SELECT * 
  FROM table_name 
  WHERE (table_name.integer_column_name > 5 
  AND table_name.integer_column_name < 7)
```

### or

One or more conditions must return `true`.

```typescript
db.select()
 .from(table)
 .where(or(gt(table.integerColumnName, 5), lt(table.integerColumnName, 7)))
 .all();
```

```sql
SELECT * 
  FROM table_name 
  WHERE (table_name.integer_column_name > 5 
  OR table_name.integer_column_name < 7)
```
