**For the quick start example we're going to use MySQL2 driver**

#### Install the dependencies
<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab>
```bash copy
npm i drizzle-orm mysql2
npm i -D drizzle-kit
```
  </Tab>
  <Tab>
```bash copy
pnpm add drizzle-orm mysql2
pnpm add -D drizzle-kit
```
  </Tab>
  <Tab>
```bash copy
yarn add drizzle-orm mysql2
yarn add -D drizzle-kit
```
  </Tab>
</Tabs>

#### Declare your first schema file
```typescript filename="schema.ts"
export const users = mysqlTable('users', {
  id: serial('id').primaryKey(),
  fullName: text('full_name'),
  phone: varchar('phone', { length: 256 }),
});
```

#### Run your first query, that's it!
```typescript copy filename="db.ts"
import { drizzle } from 'drizzle-orm/postgres-js';
import postgres from 'postgres';
import { users } from './schema';


const client = await mysql.createConnection({
  host: 'localhost',
  user: 'root',
  database: 'test',
});

const db = drizzle(client);

const allUsers = await db.select().from(users);
```
